# Beyond compliance: Building accessibility into quality with test automation

I’m hyped to share a new deep-dive article I’ve co-authored with [Jörg Jakoby](https://www.linkedin.com/in/j%C3%B6rg-jakoby-93a6a8264/), now live on the Atos blog!

We’ve been exploring how to make accessibility a "business as usual" part of the development lifecycle. The secret? Don't reinvent the wheel.

In this post, we dig into:
- How to start checking accessibility directly in your CI pipeline.
- Why automation is a massive lever for preventing regressions (even if it doesn't cover 100%).
- Practical ways to extend the tools you already have in place to save time and effort.

Building inclusive tech doesn't have to be complex and expensive. It’s about working smarter with the automation you’ve already built!
